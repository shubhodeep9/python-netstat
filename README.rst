python-netstat
==============

Ubuntu Indicator to display net usage and speed on Unity taskbar.
https://pypi.python.org/pypi/netstat-cl

Requirements:
-------------

1. Ubuntu OS
2. Python 2.6 or higher

Install
-------

::

	$ pip install netstat-cl

To Run
~~~~~~

::

    $ netstat-cl
    // To run in without interupt (background) $ netstat-cl &

To stop
~~~~~~~

::

    $ killall netstat-cl
